# ESLint v10 Migration Radar Commercial Case

## Boss-Approval Sentence

This pack tells us which repos will break or need manual work under ESLint v10,
with exact file findings, source links, and safe cleanup patches for obsolete
flags.

## Why This Can Be Paid

- ESLint v10 removed a widely used legacy config system.
- Many teams have config spread across files, package metadata, scripts, and
  tool API usage.
- The scanner is local and fast, making it useful for platform teams.

## Paid SKU Candidate

`ESLint v10 Repo Readiness Pack`

- local CLI
- JSON and HTML report
- safe script patch mode
- expanded plugin/shareable-config compatibility checks
- multi-repo summary
- rule-pack updates for v10.x migration changes

## Do Not Sell As

- complete flat-config migration
- plugin compatibility guarantee
- hosted lint service
- replacement for running ESLint tests
