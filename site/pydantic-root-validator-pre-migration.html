Pydantic issue page

Fix supported root_validator(pre=True) migrations for v2.

This page is for the narrow supported pre root-validator subset: exact two-parameter classmethod roots that can be rewritten to @model_validator(mode="before") without pretending to cover the broader deprecated surface.

  • Pack details
  • Also see validator rewrites
  • Also see BaseSettings

What changes

Before
@root_validator(pre=True)
def normalize_payload(cls, values):
    return values

After
@model_validator(mode="before")
@classmethod
def normalize_payload(cls, values):
    return values
