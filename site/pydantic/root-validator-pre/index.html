How to replace @root_validator(pre=True) with @model_validator(mode="before")

Migrate the supported pre root-validator subset in Pydantic v2 and keep the rest out of the automatic path.

Start here

The safe first move

For supported pre=True root validators with the simple (cls, values) signature, the v2 replacement is @model_validator(mode="before").

Stop before automation when: Post root validators. root_validator kwargs outside pre=True.

Target shape

from pydantic import BaseModel, model_validator

class Payload(BaseModel):
    name: str

    @model_validator(mode="before")
    def normalize_input(cls, values):
        return values

What changes

For supported pre=True root validators with the simple (cls, values) signature, the v2 replacement is @model_validator(mode="before").

Example

Before and after

Before

from pydantic import BaseModel, root_validator

class Payload(BaseModel):
    name: str

    @root_validator(pre=True)
    def normalize_input(cls, values):
        return values

After

from pydantic import BaseModel, model_validator

class Payload(BaseModel):
    name: str

    @model_validator(mode="before")
    def normalize_input(cls, values):
        return values
