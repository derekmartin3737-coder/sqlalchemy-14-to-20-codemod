Pydantic Migration Guides

Fix Pydantic v2 @validator deprecation warnings

Replace simple @validator usage with @field_validator in the safe Pydantic v2 subset.

Search intent

PydanticDeprecatedSince20 validator field_validator

Simple validators are safe cleanup; signature-heavy validators need review.

Direct answer

What changes

Use @field_validator for simple field validators, but stop when the validator depends on values, field, config, each_item, or always.

Example

Before and after

Before

from pydantic import BaseModel, validator

class User(BaseModel):
    @validator("email")
    def clean_email(cls, value):
        return value.lower()

After

from pydantic import BaseModel, field_validator

class User(BaseModel):
    @field_validator("email")
    @classmethod
    def clean_email(cls, value):
        return value.lower()
