SQLAlchemy issue page

Fix legacy select([columns]) calls before they slow the whole upgrade down.

The list-style select() syntax is one of the classic SQLAlchemy 2.0 migration chores. This page is for teams trying to qualify whether the fix is isolated or part of a bigger supported cleanup set.
