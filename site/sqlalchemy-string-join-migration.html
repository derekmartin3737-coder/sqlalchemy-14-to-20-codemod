SQLAlchemy issue page

Fix session.query(...).join("addresses") without manually tracing every relationship.

This page targets the supported simple string relationship join path: when the root entity is obvious, the pack can qualify and rewrite the old string join into an attribute-based join.

What changes

Before
session.query(User).join("addresses")

After
session.query(User).join(User.addresses)
