Well of the Month

GitHub Actions Upgrade Guard

The first Zipper Tools Product Well: a local GitHub Actions upgrade scanner for workflow deprecations, runner drift, permissions risk, and patch previews.

Current Product Well

GitHub Actions workflow upgrades have deadline-shaped pain.

Action Guard turns artifact/cache deprecations, runner-image drift, runtime pressure, token-permission risk, and invalid YAML into a local report plus safe patch preview.

Run free Action Guard scanner Read proof Open product page

Why this well exists

  • Workflow deprecations and runner changes are deadline-sensitive but scattered across repos.
  • Teams need patch previews and risk summaries, not a vague upgrade blog post.
  • The scanner can work from local workflow YAML without GitHub credentials.

Current launch boundary

  • Free scanner/report path is live in the public repo.
  • Proof artifacts are public and source-linked.
  • Paid checkout is paused until the SKU and demand signal are clear.

Rules in the first pack

  • Public proof shows deprecated artifact and cache actions detected with a patch preview.
  • Runner, Node runtime, permission, local-action, and invalid-YAML cases stay visible as findings instead of guessed fixes.
  • The scanner runs locally with no GitHub token, source upload, or repo mutation unless apply mode is explicitly used.
